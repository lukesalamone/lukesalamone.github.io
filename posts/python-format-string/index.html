How to use python f-strings

There are three main ways to format strings in python:

name = 'Luke'
food = 'pizza'

# old style
"My name is %s and I like %s." % (name, food)

# str.format()
"My name is {0} and I like {1}.".format(name, food)

# f-strings
f"My name is {name} and I like {food}."
