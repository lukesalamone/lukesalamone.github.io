Managing Python Environments

Need to switch between python versions often? Use pyenv.

Installing pyenv

Run curl https://pyenv.run | bash from terminal to install pyenv.

Check it’s installed correctly by running which pyenv.

Install another python version

Run pyenv install --list to see a list of available python versions. Once you’ve chosen a version install it like pyenv install 3.7.5 by substituting the version you want for 3.7.5.

List installed python versions

Run pyenv versions to check installed versions.

Switch python versions

Use pyenv global 3.7.5 to use python 3.7.5 everywhere on your machine.

Use pyenv local 3.7.5 to use python 3.7.5 in the current directory.

Use pyenv shell 3.7.5 to use python 3.7.5 in the current shell session.
